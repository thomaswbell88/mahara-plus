Plus
======================

> Mahara ePortfolios theme, taking inspiration from Google+.

## Current features:

 * "Small Page Headers" styled above 900px
 * Compact Page navigation
 * G+ like interface
## Tested compatibility:
 * Chrome 25+
 * IE7
 * IE8
 * IE10
 * Firefox 21+
